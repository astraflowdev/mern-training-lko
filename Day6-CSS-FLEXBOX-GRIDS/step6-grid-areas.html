Step 6: Grid Areas & Item Placement

Learn how to place grid items using line numbers, spans, and named grid areas.

1. Grid Lines - Manual Placement

Place items using grid line numbers

A (spans 2 columns)
B (spans 2 rows)
C
D
/* Item A */ grid-column: 1 / 3; /* Start at line 1, end at line 3 (spans 2 columns) */ /* Item B */ grid-row: 1 / 3; /* Start at line 1, end at line 3 (spans 2 rows) */

2. Using Span Keyword

Easier way to specify how many columns/rows to span

Spans 2 Columns
Spans 3 Columns
Normal
grid-column: span 2; /* Span across 2 columns */ grid-column: span 3; /* Span across 3 columns */

3. Grid Template Areas - Named Regions

Create a layout by naming areas - very intuitive!

Header
Sidebar
Main Content
Footer
/* Container */ grid-template-areas: "header header header" "sidebar content content" "footer footer footer"; /* Items */ .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: content; } .footer { grid-area: footer; }

Visual ASCII-art style layout definition!

4. Complex Dashboard Layout

More complex layout with multiple named areas

Header/Nav
Sidebar
Main Content
Widget 1
Widget 2
Footer
grid-template-areas: "header header header header" "sidebar main main widget1" "sidebar main main widget2" "footer footer footer footer";

5. Empty Cells with Dot Notation

Use dots (.) to leave cells empty

Header
Content (with empty space around)
Footer
grid-template-areas: "header header header" ". content . " "footer footer footer"; /* The dots (.) create empty cells */

6. Aligning Items within Grid Cells

Control how items align within their grid cells

Start
Center
End
Stretch
justify-items: start; /* Horizontal alignment */ align-items: center; /* Vertical alignment */ /* Or on individual items */ justify-self: end; align-self: start;

7. Overlapping Grid Items

Items can overlap - use z-index to control stacking

Background
Foreground (overlaps)
/* Both items occupy the same grid cells */ .overlap-bg { grid-area: 1 / 1 / 3 / 3; z-index: 1; } .overlap-front { grid-area: 2 / 2 / 4 / 4; z-index: 2; }

Key Takeaways:

  • grid-column: 1 / 3 - Place item from column line 1 to 3
  • grid-row: 2 / 4 - Place item from row line 2 to 4
  • grid-column: span 2 - Span 2 columns (easier than line numbers)
  • grid-template-areas - Create named regions for intuitive layouts
  • grid-area: name - Assign element to a named area
  • . (dot) - Leave cells empty in template areas
  • justify-items / align-items - Align all items in container
  • justify-self / align-self - Align individual items
  • z-index - Control stacking order of overlapping items
  • Grid lines are numbered starting from 1 (not 0!)
