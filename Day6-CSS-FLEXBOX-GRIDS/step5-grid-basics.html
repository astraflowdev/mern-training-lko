Step 5: CSS Grid Basics

CSS Grid is a two-dimensional layout system for creating complex, responsive layouts with rows and columns.

1. Basic Grid - 3 Columns

Creating a simple 3-column grid with equal widths

1
2
3
4
5
6
grid-template-columns: 1fr 1fr 1fr;

1fr = 1 fraction of available space

2. Different Column Widths

Columns with different proportions using fr units

1 (2fr)
2 (1fr)
3 (1fr)
4
5
6
grid-template-columns: 2fr 1fr 1fr;

First column is twice as wide as others

3. Fixed and Flexible Columns

Mixing fixed pixel widths with flexible fr units

1 (200px)
2 (flexible)
3 (150px)
4
5
6
grid-template-columns: 200px 1fr 150px;

Sidebars fixed, main content flexible

5. Grid Gap (Spacing)

Adding space between grid items using gap property

1
2
3
4

6. Different Row & Column Gaps

Setting different gaps for rows and columns

1
2
3
4
5

7. Grid Template Rows

Defining explicit row heights

1 (100px tall)
2
3 (150px tall)
4
5

8. Responsive Grid with auto-fit and minmax

Grid automatically adjusts columns based on available space

1
2
3
4
5
6
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

Automatically creates columns, min 200px, max 1fr. Try resizing window!

Key Takeaways:

  • display: grid - Creates a grid container
  • grid-template-columns - Defines column sizes
  • grid-template-rows - Defines row sizes
  • fr unit - Fraction of available space (1fr, 2fr, etc.)
  • repeat(count, size) - Repeat pattern without writing it multiple times
  • gap - Space between grid items (replaces grid-gap)
  • minmax(min, max) - Sets minimum and maximum size constraints
  • auto-fit - Automatically fits columns into available space
  • Grid is best for two-dimensional layouts (rows AND columns)
