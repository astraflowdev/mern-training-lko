Key Takeaways:

  • display: flex - Makes an element a flex container
  • flex-direction: row - Default, items in horizontal row (left to right)
  • flex-direction: row-reverse - Horizontal row (right to left)
  • flex-direction: column - Vertical column (top to bottom)
  • flex-direction: column-reverse - Vertical column (bottom to top)
