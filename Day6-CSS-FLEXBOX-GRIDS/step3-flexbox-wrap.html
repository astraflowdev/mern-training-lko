Key Takeaways:

  • flex-wrap: nowrap - All items on one line (default)
  • flex-wrap: wrap - Items wrap to new lines as needed
  • flex-grow - How much an item should grow (default: 0)
  • flex-shrink - How much an item should shrink (default: 1)
  • flex-basis - Initial size before growing/shrinking (default: auto)
  • flex: 1 - Shorthand for flex-grow: 1, flex-shrink: 1, flex-basis: 0
  • flex: 0 1 auto - Default flex shorthand value
